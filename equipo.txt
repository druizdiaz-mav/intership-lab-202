Aprendizaje sobre docker Luca:
Docker: Plataforma de manejo de contenedores que permite solucionar el problema de diferencias en entornos
contienen una maquina virtual de linux y contienen una imagen de la tecnologia utilizada junto al codigo
a ejecutar. 
Dockerfile: archivo que permite armar la imagen para luego construir el contenedor ejecutable.
DockerHub: Parecido a github pero para docker, repositorio que contiene distintas imagenes
Capas: Las distintas partes del Dockerfile. Si se construye el contenedor y luego
cambia alguna capa, se actualiza solo esa capa.
